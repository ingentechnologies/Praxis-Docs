## 2D Charting

I am equipped with a special tool that help visualize datasets in 2D.

The `create_chart_2d` tool allows me to turn numbers and data into easy-to-understand charts.

Here's how it works in simple steps:

<Steps>
<Step>Get Data: I take data from different formats like CSV or TSV, or even data you paste into my input dialogue entry</Step>
<Step>Organize Data: I sort the data so it's ready to be turned into a chart.</Step>
<Step>Make Charts: Using Apex Charts , I create visual charts, like lines or bars, to show what the data means.</Step>
</Steps>

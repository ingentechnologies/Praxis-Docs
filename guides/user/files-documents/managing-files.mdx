## Adding Files to Your IP Vault

<Steps>
  <Step title="Access Your File Manager">
    Click the **Files** button on the text input bar to open your IP Vault.

    ![Files Location Main Ui Gi](/images/files-location-main-ui.gif)
  </Step>
  <Step title="Start Adding Content">
    In your IP Vault dashboard, click the **\+ Add** button at the bottom of the page.

    <img
      src="/images/files-ip-vault-landing.webp"
      alt="Files Ip Vault Landing Web"
      title="Files Ip Vault Landing Web"
      style={{ width:"64%" }}
    />
  </Step>
  <Step title="Choose Your Upload Method">
    ![Files Add Options Web](/images/files-add-options.webp)

    Select from three convenient options:

    1. **Device Upload** - Browse and select files from your computer
    2. **URL Import** - Import content directly from a web page
    3. **Clipboard Paste** - Copy and paste content directly
  </Step>
  <Step title="Complete the Upload">
    After selecting your files, click **Upload** to begin the process.

    ![Files Upload Button Example Web](/images/files-upload-button-example.webp)

    <Note>
      A progress indicator will show upload status. File processing (ingestion) may take several minutes depending on file size and type.
    </Note>
    ![Files Upload Processing Bar Web](/images/files-upload-processing-bar.webp)

    When complete, your file status will display as **"Success"** and be ready for use. Click **Done** in the lower-left corner to go back to your IP Vault.

    ![Files Upload Success Web](/images/files-upload-success.webp)
  </Step>
  <Step title="Verify Files in Your Vault">
    Your uploaded files now appear in your IP Vault, ready for your digital twin to reference.

    ![Files In Ip Vault Web](/images/files-in-ip-vault.webp)
  </Step>
</Steps>

## Supported File Formats

Your IP Vault accepts a comprehensive range of file types:

| **Content Type**     | **Supported Formats**        |
| -------------------- | ---------------------------- |
| **Images**           | PNG, WEBP, JPEG, JPG         |
| **Audio**            | M4A, MP3, WAV, WEBM          |
| **Video**            | MP4, WEBM, MPEG              |
| **Documents**        | PDF, DOC, DOCX, ODT          |
| **Spreadsheets**     | XLS, XLSX, ODS, CSV, TSV     |
| **Presentations**    | PPT, PPTX, ODP               |
| **Web Content**      | HTML, HTM, MHTML, XHTML      |
| **Text & Data**      | TXT, JSON, XML               |
| **Development**      | IPYNB (Jupyter Notebooks)    |
| **Learning Content** | IMSCC (IMS Common Cartridge) |
| **Archives**         | ZIP                          |

<Warning>
  Maximum upload limit: 500MB total across all files
</Warning>

### File Actions Explained

- <Icon icon="pencil" color="#0e84e4" /> **Edit** - Update file names and descriptions for better organization
- <Icon icon="eye" color="#f88820" /> **Details** - View comprehensive file information and processing status
- <Icon icon="toggle-large-on" color="#122ae1" /> **Include** - Enable/disable your digital twin's access to this file
- <Icon icon="toggle-large-on" color="#122ae1" /> **Confidential** - Mark files as private (usable internally but not cited publicly)
- <Icon icon="link" color="#48a62e" /> **Share** - Generate shareable URLs for collaboration
- <Icon icon="copy" color="#aba5a5" /> **Copy** - Copy file names to clipboard
- <Icon icon="arrow-down-to-bracket" /> **Download** - Save copies to your device
- <Icon icon="arrows-rotate" color="#f07704" /> **Refresh** - Reload files to update dynamic content
- <Icon icon="trash-can" color="#df1010" /> **Delete** - Permanently remove files from your vault

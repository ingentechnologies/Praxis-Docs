<Tip>
  You only need to select **one** integration method. For optimal performance, we recommend the **Custom Theme** approach.
</Tip>

<CardGroup cols={2}>
  <Card title="Custom Theme" icon="palette" href="https://docs.praxis-ai.com/integrations/canvas/theme">
    - This option allows institution admins to add Praxis AI as an overlay to all pages within an account or sub-account using a specific Canvas theme.
    - 
    - Students access Praxis AI as a floating icon in the corner of their course windows.
  </Card>
  <Card title="LTI 1.3 Activity" icon="link" href="https://docs.praxis-ai.com/integrations/canvas/lti-1-3">
    - This option allows faculty to add Praxis AI as an activity in specific modules within specific courses.
    - 
    - Students access Praxis AI as a web application in a new browser tab.
  </Card>
</CardGroup>
